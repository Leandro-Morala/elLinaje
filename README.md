# El Linaje, Hijos del Dios altimiso.
-> El proposito es que el usuario pueda tener 
    una relacion mas cercana a Dios, y gradualmente, 
    poder orar mas, una comprencion de las escrituras mas profunda, 
    tener responsabilidad financiera, hacer benefisencia en la iglesia 
    y ver los beneficios que hace tener el servicio a Dios.

-> Para alcanzar estos objetivos la aplicacion se dispone en modulos que 
    desafian al usuarios con retos, de oracion por dia, 
    lectura de la palabra por memorizacion, donaciones responsables, 
    colaboracion en la iglesia por tiempos y muestreos.

-> Tambien cuenta con una funcion de biblia escondida, que el usuario
    puede consultar para instruirse y crear su propio estudio biblico
    si asi lo desea. 
    
-> Futura expacion: competencia de equipo, poder crear perfil en linea
    con informacion ( restringida por el usuario ) para que
    participantes elegidos y autorizados se integren al grupo 
    y puedan participar en competencias con sus capacidades.
